Image Rotator

Super Simple jQuery Image Rotator Plugin

Image Rotator is a simple jQuery plugin that allows you to take collection of images and easily transform them into an easy to style interactive slideshow.

How It Works

Couldn't be simpler. Include the plugin and tell jQuery what to do.

  1. 1<script src="jquery.js"></script>
  2. 2<script src="jquery.imagerotator.js"></script>
  3. 3<script>
  4. 4 $(document).ready(function(){
  5. 5 $("#imgRotator").imgRotate({width:400,
  6. 6 height:300});
  7. 7 });
  8. 8</script>

Plugin Options

Take a look at the various options the plugin provides. Defaults are included next to the option name.

  1. 1width: 400
  2. 2height: 300
  3. 3speed: 500
  4. 4autoRotate: true
  5. 5autoRotateDelay: 2000
  6. 6showArrows: true
  7. 7showCircles: true
